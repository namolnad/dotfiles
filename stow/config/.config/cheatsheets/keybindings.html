Keybindings Cheatsheet

How contexts work: AeroSpace uses binding modes to isolate workspaces. Press Alt+1-4 to switch context. Per-context keys (T, B, E, C) map to context-prefixed workspaces (e.g. Alt+T in Work mode → W-T). Add Shift to any workspace key to move the focused window there.
